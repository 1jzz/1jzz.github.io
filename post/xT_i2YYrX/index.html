idea远程调试

2022-04-12 1 min read # 环境

idea配置Remote

  1. 添加一个Remote JVM Debug
  2. 随便改个名字和选择一个空闲端口
  3. 配置调试中jdk的版本
  4. 选择使用模块类路径(想要调试项目的代码)
  5. 复制启动参数

-agentlib:jdwp=transport=dt_socket,server=y,suspend=n,address=5005

jar包远程调试

  1. 复制Remote中的启动参数
  2. 以该启动参数运行jar包
  3. 即可在idea中下断点调试

java -jar -agentlib:jdwp=transport=dt_socket,server=y,suspend=n,address=5005

war包远程调试

  1. 复制Remote中的启动参数
  2. 修改tomcat下cateline.bat或cateline.sh中的JAVA_OPTS
  3. 以cateline.bat为例

set "JAVA_OPTS=%JAVA_OPTS% %JSSE_OPTS% -Dfile.encoding=UTF-8 -agentlib:jdwp=transport=dt_socket,server=y,suspend=n,address=5005"
